HANDOFF NOTE - CV32E40P + HPDCache D-Cache Integration
Date: 2026-05-26

1. Project context
------------------
This project integrates an L1 D-Cache into the CV32E40P RISC-V core.

Current direction:
- Use CV32E40P as the main core.
- Focus only on D-Cache first.
- Do not integrate I-Cache yet.
- D-Cache is based on HPDCache modified with write-back + VBUF.

Target architecture:

CV32E40P instruction port
  -> direct instruction memory model

CV32E40P data port
  -> cv32_data_to_hpdcache_adapter
  -> HPDCache write-back + VBUF
  -> data memory model

5. How to run in Ubuntu
-----------------------
Use this from Ubuntu:

ROOT_WORKSPACE=$(find /media -maxdepth 8 -type d -name "cv32e40p-master" 2>/dev/null | grep "/work/" | head -n 1)
cd "$ROOT_WORKSPACE/rtl/l1_cache/work/sim" || exit 1

Run build:

make build

Run main integrated D-Cache basic test:

make run

Run baseline no-cache:

make run_baseline

Run adapter basic:

make run_adapter_basic

Run random smoke seeds 1/2/3:

make run_cv32_dcache_random

Open main waveform:

make wave_basic

Or directly:

gtkwave "$ROOT_WORKSPACE/rtl/l1_cache/work/waves/03_cv32_dcache_basic.vcd" \
        "$ROOT_WORKSPACE/rtl/l1_cache/work/waves/dcache_basic.gtkw" &
